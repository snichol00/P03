Scala

Overview

Scala stands for "scaleable language," meaning that it can be scaled up or down to meet the user's needs, using a combination of object-oriented and functional programmins. Object-oriented programming will be familiar to java users, and includes the four key tenets of polymorphism, encapsulation, inheritance, and abstraction. In many ways, Scala is a more concise and intuitive version of Java, but its functional programming aspect provides more substantive differences.

Key Features

  • Object-oriented: Modularity makes it easier to troubleshoot
  • Statically typed: Catches errors at compile time, before execution
